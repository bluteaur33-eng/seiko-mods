Configuration via Google Apps Script
1

Créer un Apps Script

Va sur script.google.com → Nouveau projet → Colle le code du webhook (fourni séparément) → Remplace le contenu du fichier Code.gs.

2

Déployer comme application web

Déployer → Nouveau déploiement → Type : Application Web → Exécuter en tant que : Moi → Accès : Tout le monde → Déployer → Autorise l'accès → Copie l'URL.

3

Coller l'URL du webhook ici

L'URL ressemble à https://script.google.com/macros/s/…/exec
